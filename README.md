# 🚀 Asp .Net Core MVC ile Admin Panelli Kafe Sitesi Projesi

Bu proje, ASP.NET Core MVC ve Code First yöntemi kullanılarak geliştirilmiş admin panelli dinamik bir kafe sitesidir. Site içerisinde kafe ile ilgili görseller, zengin bir içerik barındıran menü, referanslar ve dinamik bir abonelik istek alanı mevcuttur. Kullanıcıların istek ve önerilerini iletebilecekleri; mesajların admin paneli ve veritabanına başarıyla kaydedildiği dinamik bir contact alanı kullanıcıları karşılamaktadır. 
Portföy sitesi verilerinin yönetimi için kullanıcı dostu bir admin paneli de projede yer almaktadır.
